# JAVAISLAMI - Modern Islamic Web App
Aplikasi panduan Islam modern dengan desain minimalis (Blue & White).

## Fitur
- Jadwal Sholat Real-time (GPS Based)
- Al-Qur'an Digital & Progress Reading
- Komunitas Bisnis Syariah
- Arah Kiblat Presisi

## Tech Stack
- HTML5, CSS3 (Custom Properties), JavaScript ES6
- Deployment: Vercel / GitHub Pages
